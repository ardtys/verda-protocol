technical

Under the Hood

QR Code Structure

Each code encodes a unique product identifier (UPI) using a compressed format. Error correction level H ensures readability even when partially damaged. 

verda://p/[company-id]/[product-id]/[batch]/[serial]
Example: verda://p/VRD001/COFFEE-01/B2024-0315/00001234

Verification Flow

  • Scan triggers lookup against Verda API
  • Server returns journey data + Merkle proof
  • Client verifies proof against Blockchain state
  • Results cached locally for offline access
